## 자주 묻는 질문 (FAQ)

### Q1. 추천 결과가 마음에 들지 않아요. 어떻게 하나요?

추천 화면에서 **"다른 추천 보기"** 버튼을 눌러주세요. 새로운 추천이 나타납니다.
또한 **"별로예요"** 버튼을 누르시면 다음부터 비슷한 추천이 나오지 않도록 학습됩니다.

### Q2. 위치 권한을 거부했는데 추천이 안 돼요.

'착'은 주변 식당을 찾기 위해 위치 정보가 필요합니다.
- iOS 설정 → 개인정보 보호 및 보안 → 위치 서비스 → 착(CHAK) → 권한 허용

위치 정보는 추천 요청 시점에만 사용되며, 서버에 영구 저장되지 않습니다.

### Q3. 알러지나 못 먹는 음식을 등록하고 싶어요.

설정 메뉴에서 **"내 취향 관리"** 를 통해 등록할 수 있습니다.
- 알러지 항목
- 절대 먹지 않는 음식
- 선호하는 음식

등록한 정보는 추천 결과에 즉시 반영됩니다.

### Q4. 회식 모드는 어떻게 사용하나요?

홈 화면에서 **"회식"** 모드를 선택한 뒤 다음 정보를 입력해 주세요.
- 인원수
- 1인당 예산
- 분위기 (조용한, 활기찬, 트렌디 등)

입력 후 '회식 장소 찾기'를 누르면 조건에 맞는 식당이 추천됩니다.

### Q5. 회원 탈퇴는 어떻게 하나요?

설정 메뉴에서 **"회원 탈퇴"** 를 누르시면 즉시 처리됩니다.
- 모든 개인정보(취향, 저장한 식당, 계정 정보)는 즉시 삭제됩니다.
- 한번 탈퇴 시 데이터 복구가 불가능합니다.

이메일을 통한 탈퇴 요청도 가능합니다 (for.ps.project@gmail.com).

### Q6. 로그인이 안 돼요.

다음을 순서대로 확인해 주세요.
1. 인터넷 연결 상태
2. 이메일/비밀번호 정확히 입력했는지
3. Google 로그인 사용 시, Google 계정 정상 작동 여부
4. 앱을 완전히 종료 후 재실행

문제가 지속되면 위 이메일로 연락 주세요.

### Q6. 추천 결과 식당 정보가 실제와 달라요.

식당 정보는 외부 서비스(Google Places, Kakao Local)에서 제공받습니다.
영업 시간, 메뉴, 가격 등은 변동될 수 있으므로 방문 전 직접 확인을 권장드립니다.
